# Resume

React project using TypeScript developed during Campinas Tech Talents bootcamp.

## Getting started

**Installing**
>Cloning the repository

```bash
$ git clone https://github.com/biagavirete/curriculo-ts.git
$ cd curriculo-ts
```

**Running**
> Installing dependencies

```bash
$ npm install
```

> Running React

```bash
$  npm start
```
