# Currículo - TS

Projeto React utilizando Typescript desenvolvido na trilha React do Campinas Tech Talents.

## Como usar

**Para instalar**
> Clonar o repositório

```bash
$ git clone https://github.com/biagavirete/curriculo-ts.git
$ cd curriculo-ts.git
```

**Para rodar**
> Instalar dependências

```bash
$ npm install
```

> Iniciar o React

```bash
$  npm start
```
